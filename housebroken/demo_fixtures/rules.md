# Night-Shift Agent — House Rules

A tiny, fully synthetic rules file for `housebroken demo`. It tells the story of a
hapless coding agent working the night shift and breaking curfew.

## Curfew

- Never use the `--force` flag when pushing to a shared branch.
- Never use the `--hard` flag with git reset.

## Commits

- Use conventional commit messages.
- Run the test suite before committing.

## Craft

- Prefer small, surgical changes over sweeping rewrites.
- Write clean, readable code.

## Legacy

- Never use the `--no-cache` flag against the staging database.

## Vibes

- Always be excellent.
